This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Run the development server:

```bash
npm run dev
```

Check the code for standard compliance.

```bash
npm run lint
```

Fix the scss code automatically.

```bash
npm run stylelint
```

## Folder Structure
- `/public`: This folder contains all static files: fonts, icons etc.
- `/components`: This folder contains ui components.
- `/pages`: All the pages created here which is used by Next.js routing mechanism.
- `/store`: This folder contains all necessary for Redux toolkit.
- `/services`: It's contain all the data fetching related codes.
- `/layouts`: This folder contains ui layouts.
- `/types`: This folder contains global TypeScript types.
- `/styles`: Here is global css code and plugins' code.
- `/hooks`: Custom hooks are here.
- `/hocs`: HOC stands for High order components. HOCS are here.
- `/utils`: It contains all global project constants and help functions.
